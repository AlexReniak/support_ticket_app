# support_ticket_app

## About
Users can post a support ticket for Apple products and talk with an admin to  resolve the issue.

## Technologies Used
Support ticket app is built using the following technologies:

- React
- Redux
- React Router
- CSS
- Node
- Express
- MongoDB
